Ad alta voce - Podcast non ufficiale

Script per la generazione dei Feed Podcast da Ad Alta Voce, il programma di Radio 3.

Ultimo aggiornamento: 2021-07-05.

  • Lista dei podcast

Istruzioni

Installazione

Clonare il repository

git clone https://github.com/timendum/adaltavoce.git

Installare le dipendenze

pip install -r requirements.txt

Utilizzo

Per generare un singolo Feed podcast:

python single.py https://www.raiplayradio.it/playlist/20...

Per generare tutti i feed in un colpo solo nella cartella dist.

python all.py -nohtml

L'opzione -nohtml evita la creazione di questa pagina HTML.
